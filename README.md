# accBook
졸업 작품 - 간단 가계부 어플리케이션입니다.
NotificaionListenerService 를 이용해 카카오톡 푸시알림 텍스트를 읽어와,결제나 상대에게 입금받을 시 
RecyclerView에 입금과 지출이 자동으로 출력됩니다.
그 달의 총 지출과 수입을 한 눈에 볼 수 있고, 용도에 따른 자세한 금액을 Reciept으로 확인할 수 있는 간단한 가계부 어플리케이션입니다.
